This repository contains the original source code of the [Source SDK 2013](https://github.com/ValveSoftware/source-sdk-2013) (Valve) with some improvements.

## About

This project provides some improvements from our old modifications to the original **Half-Life 2** (Base Level) within the framework of the latest current version of **Source SDK 2013**, designed to work with **Source SDK Base 2013 Multiplayer**.

Information about the changes is provided in the [CHANGES](https://github.com/rabbitGraned/source-sdk-2013/blob/master/CHANGES.md) file.

## Requirements

> The requirements are duplicated from the original repository and are current at the time of December 2025.

### Windows:

 - Source SDK 2013 Multiplayer (Steam version)
 - Visual Studio 2022 with the following workload and components:
   - Desktop development with C++:
     - MSVC v143 - VS 2022 C++ x64/x86 build tools (Latest)
     - Windows 11 SDK (10.0.22621.0) or Windows 10 SDK (10.0.19041.1)
 - Python 3.13 or later

Testing and debugging were carried out within Visual Studio 2022 (with all the listed components) and Python 3.14.
